# camdangel
Camdan Gel Project

## Used Technologies
- Web Api: [.NET Core 3.1](https://github.com/dotnet/core)
- Web App: [Angular 9](https://github.com/angular/angular)


## How to use projects

### For Web Api

```
cd camdangel.webapi
dotnet run
```

### For Web App

```
cd camdangel.web/camdangel
ng serve
```
- Open Browser With Url: localhost:4200
